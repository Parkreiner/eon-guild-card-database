# Database Design

## Fields

1. Row number
2. Card region
3. Card number for that region
4. Character name
5. Current level
6. Retirement level (can be `Null`)
7. Main class
8. Sub-class (can be `Null`)
9. Custom title (can be `Null`)
10. Guild name
11. Community name (can be `Null`)
12. URL to guild card image
13. Upload date
